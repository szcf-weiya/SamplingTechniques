# 整群抽样

## 定义

将总体划分为若干群,以群为抽样单元,从总体中随机抽取一部分群,对入选群内的所有单元进行调查的一种抽样技术.

## 原则
群内差异尽可能大,群间差异尽可能小

## 规模
1. 相等
2. 不相等
