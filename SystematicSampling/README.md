# 系统抽样

## 定义与实施方法

### 什么是系统抽样

1. 总体中的N个单元按一定顺序排列,
2. 抽取一个起始单元,
3. 按某种规则选取其它单元直到满n个为止

### 抽样类型

1、按无关标志排列
2、按有关标志排列
3、自然排列

## 等概系统抽样

$N\neq nk$，由于$k$个可能样本所包含的单元数不全相等，因此$\bar y_{sy}$是有偏的。

三种改造为无偏估计量的方法
1. 采用圆形抽样系统
2. 采用直线抽样系统
3. 起始点仍在 1 ~ k 范围内随机产生
